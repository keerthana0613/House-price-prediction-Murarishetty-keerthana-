# Charts Folder
This folder contains visualizations created for the House Price Prediction project.

## Charts Included
 1. price_distribution.png
Histogram showing the distribution of house prices in the dataset. It helps identify the spread of prices and any skewness in the data.
 2. correlation_heatmap.png
Correlation heatmap displaying the relationships between features and house price. Strong positive or negative correlations help identify influential variables.
 3.actual_vs_predicted.png
Scatter plot comparing actual house prices with model predictions.
 4. feature_importance_treemap.png
Treemap visualization of the top features affecting house prices based on the Random Forest Regressor model. Larger blocks indicate features with greater importance in predicting house prices.

## Purpose
These visualizations were created to:
- Explore the distribution of house prices.
- Understand relationships between variables.
- Identify the most important factors influencing house prices.
- Support model evaluation and business insights.
